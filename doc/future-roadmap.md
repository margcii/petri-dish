# Petri Dish 后续开发清单

## 已规划功能

### 1. 三个真菌杂交机制 ⭐ 高优先级

**需求描述**：当前系统只支持两个真菌杂交，需要扩展支持三个真菌同时杂交

**业务逻辑**：
```
当前（两个真菌）：[🟦] + [🟨] → AI融合
目标（三个真菌）：[🟦] + [🟨] + [🟥] → AI融合

显示方式：
- 两个真菌：横向平行重叠50%
- 三个真菌：品字形重叠50%
```

**技术实现**：
- [ ] 后端：修改 `/trigger_hybrid` API 支持 3 个真菌参数
- [ ] 后端：调整 AI 融合 prompt，支持三个文本融合
- [ ] 前端：修改自动检测算法（当 ≥3 个 idle 真菌时，可选择 3 个杂交）
- [ ] 前端：品字形显示已实现，需验证三个真菌时的交互

**数据库变更**：无（parent1_id, parent2_id 已足够，或添加 parent3_id）
